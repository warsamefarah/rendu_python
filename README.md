# Rendu Python/Pygame

Jeu de casse-briques réalisé dans le cadre du module Python.

_____
### Comment lancer le jeu

* Ouvrez un terminal/invite de commande dans ce dossier
*  Tapez la commande ```python game.py```

___
### Comment jouer

* **&#x2191;** et **&#x2193;** pour bouger légèrement vers le haut et le bas.
* **&#x2190;** et **&#x2192;** pour bouger à gauche et à droite.
    * Gardez **ESPACE** enfoncé pour bouger plus vite à gauche et à droite.
* **P** pour faire pause (réappuyez pour reprendre).
* **ÉCHAP** à tout moment pour quitter le jeu.
